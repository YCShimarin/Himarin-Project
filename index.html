Apa itu Himarin Project?

Himarin Project merupakan kumpulan portofolio dan project jangka mandiri yang saya lakukan untuk mengembangkan diri saya baik secara keterampilan ataupun lainnya


Project ini merupakan serangkaian kegiatan mandiri yang berisi portofolio pribadi saya dan beberapa resume pelajaran yang telah saya pelajari baik di kampus ataupun diluar kampus saya


Hasil dari Project ini dapat berupa desain grafis, code, website, UI design, artwork atau galeri seni pribadi saya dan ringkasan catatan saya
